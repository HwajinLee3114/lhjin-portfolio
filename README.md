### 주요 명령어

```bash
	# 개발 서버 실행 (Next.js)
	pnpm dev

	# Vite 개발 서버 실행 (테스트/실험용)
	pnpm start:local

	# 빌드
	pnpm build

	# 코드 포맷
	pnpm format

	# 코드 포맷 검사
	pnpm format:check

	# 린트
	pnpm lint

	# 테스트
	pnpm test
```
